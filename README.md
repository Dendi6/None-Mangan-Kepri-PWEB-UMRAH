## Mangan Kepri

Mengan Kepri merupakan website yang di kembangkan demi kepentingan tugas akhir Pemograman WEB. 
Tugas di kerjakan oleh Siti Marliana dengan nim 180155201016. Tugas di bangun menggunakan php, dan codeIgniter 4 sebagai framework.<br>Dan SB Admin 2 sebagai template admin

Mangan Kepri => Makanan Kepualauan Riau

## Running Code

1. Download kode dengan cara melakukan perintah
   > git clone https://github.com/Dendi6/None-Mangan-Kepri-PWEB-UMRAH.git
2. Import database,
3. Lakukan perintah jalankan seting coneksi, dengan cara merename `env` ke `.env`
4. Melakukan update composer
   > composer update
5. Running port dengan cara
   > php spark serve
